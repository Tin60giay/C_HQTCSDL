# HDSD — Hướng Dẫn Sử Dụng Phần Mềm Quản Lý Điểm Tín Chỉ PTIT
**Phiên bản:** v1.0 · **Ngày:** 2026-04-17

---

## 1. Cài Đặt & Khởi Chạy

### Yêu cầu
- Python 3.9+
- SQL Server Express (localhost\SQLEXPRESS)
- Database `QLDSV_HTC` đã tồn tại

### Cài đặt thư viện
```bash
pip install flask pyodbc
```

### Chạy setup SQL (1 lần duy nhất)
Mở SQL Server Management Studio, chạy file `setup_login.sql` để tạo tất cả stored procedures và SQL logins.

### Khởi chạy ứng dụng
```bash
cd C_HQTCSDL
python app.py
```
Truy cập: http://localhost:5000

## 3. Hướng Dẫn Sử Dụng Theo Nhóm

### 3.1 Nhóm PGV (Phòng Giáo Vụ)

#### Quản lý Lớp Cử Nhân (`/lop`)
1. Click **📋 Quản lý Lớp Cử Nhân** từ dashboard
2. **Thêm lớp:** Click **＋ Thêm** → điền Mã lớp, Tên lớp, Khóa học, Khoa → **💾 Ghi**
3. **Sửa lớp:** Click vào dòng lớp trong bảng → chỉnh sửa → **💾 Ghi**
4. **Xóa lớp:** Chọn dòng → **🗑 Xóa** (xác nhận)
5. **Phục hồi:** Hoàn tác thay đổi chưa ghi bằng nút **↩ Phục hồi**

#### Quản lý Sinh Viên (`/sinhvien`)
1. Panel trái: danh sách lớp → click một lớp
2. Panel phải: danh sách sinh viên của lớp đó
3. Thêm/Sửa/Xóa tương tự như Quản lý Lớp

#### Quản lý Môn Học (`/monhoc`)
- Thêm/Sửa/Xóa môn học (Mã MH, Tên, Số tiết LT, Số tiết TH)

#### Mở Lớp Tín Chỉ (`/loptinchi`)
1. Dùng bộ lọc (Niên khóa, Học kỳ, Khoa) để thu hẹp danh sách
2. **Mở lớp mới:** Click **＋ Thêm** → điền thông tin → **💾 Ghi**
3. **Hủy lớp:** Chọn dòng → **🚫 Hủy lớp** (lớp bị đánh dấu HUYLOP=1)

#### Nhập Điểm (`/nhapdiem`)
1. Chọn Niên khóa, Học kỳ, Môn học, Nhóm → **▶ Bắt đầu**
2. Bảng sinh viên hiện ra với các ô Điểm CC, GK, CK
3. Điểm HM tự động tính: `CC×0.1 + GK×0.3 + CK×0.6`
4. Nhập xong → **💾 Ghi điểm**

---

### 3.2 Nhóm KHOA (Giảng Viên Khoa)

- **Xem Lớp Tín Chỉ** (`/loptinchi`): Chỉ xem, không chỉnh sửa
- **Nhập Điểm** (`/nhapdiem`): Tương tự PGV
- **Báo Cáo**: (chức năng mở rộng)

---

### 3.3 Nhóm SV (Sinh Viên)

#### Đăng Ký Lớp Tín Chỉ (`/dangky`)
1. Vào **📝 Đăng ký Lớp Tín Chỉ**
2. Nhập **Niên khóa** và chọn **Học kỳ** → **🔍 Tìm lớp**
3. Danh sách LTC khả dụng hiện ra
4. Click **Đăng ký** ở lớp muốn đăng ký
5. Lớp đã đăng ký hiển thị badge **✓ Đã đăng ký**

#### Xem Bảng Điểm (`/phieu_diem`)
- Hiển thị toàn bộ điểm các môn đã học (CC, GK, CK, HM)
- Điểm HM ≥ 5 hiển thị màu xanh (đạt), < 5 màu đỏ (không đạt)
