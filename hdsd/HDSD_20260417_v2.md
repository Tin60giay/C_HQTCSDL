## 1. Cài Đặt & Khởi Chạy

### Yêu cầu
- Python 3.9+
- SQL Server Express (localhost\SQLEXPRESS)
- Database `QLDSV_HTC` đã tồn tại

### Cài đặt thư viện
```bash
pip install flask pyodbc
```

### Chạy setup SQL (bắt buộc mỗi khi có cập nhật SP)
Mở SQL Server Management Studio → Mở file `setup_login.sql` → **F5**

> File này chỉ tạo/cập nhật Stored Procedures, **không đụng vào dữ liệu hay schema**.

### Khởi chạy ứng dụng
```bash
cd C_HQTCSDL
python app.py
```
Truy cập: **http://localhost:5001**

## 3. Hướng Dẫn Sử Dụng Theo Nhóm

### 3.1 Nhóm PGV (Phòng Giáo Vụ)

#### Quản lý Lớp Cử Nhân (`/lop`)
1. Click **📋 Quản lý Lớp Cử Nhân** từ dashboard
2. **Lọc tức thì:** Chọn khoa hoặc gõ vào ô tìm kiếm — bảng tự lọc ngay, không cần bấm gì
3. **Thêm lớp:** Click **＋ Thêm** → điền Mã, Tên, Khóa học (gợi ý tự động), Khoa → **💾 Ghi**
4. **Sửa lớp:** Click dòng trong bảng → chỉnh sửa → **💾 Ghi**
5. **Xóa lớp:** Chọn dòng → **🗑 Xóa**
   - ⚠️ Không thể xóa nếu lớp còn sinh viên — hệ thống báo lỗi cụ thể
6. **Phục hồi:** Hoàn tác thay đổi *chưa ghi* trong form bằng **↩ Phục hồi**
7. **Lịch sử:** Click **🕓 Lịch sử** để xem 10 thao tác gần nhất và hoàn tác nếu cần

#### Quản lý Sinh Viên (`/sinhvien`)
1. Panel trái: danh sách lớp — click một lớp để xem SV
2. Panel phải: danh sách sinh viên + form nhập liệu
3. Thêm/Sửa/Xóa/Phục hồi/Lịch sử tương tự như Quản lý Lớp
4. Quan hệ cha–con: **SV thuộc Lớp** — nếu hoàn tác "Thêm lớp" mà đã thêm SV vào lớp đó, cần xóa SV trước

#### Quản lý Môn Học (`/monhoc`)
- Lọc tức thì theo mã/tên môn học
- **Ràng buộc:** Không thể xóa môn học nếu có sinh viên đã đăng ký lớp tín chỉ thuộc môn đó
- CRUD + Lịch sử tương tự Quản lý Lớp

#### Mở Lớp Tín Chỉ (`/loptinchi`)
1. **Bộ lọc tức thì 4 chiều:** Chọn Niên khóa / Học kỳ / Khoa / gõ tên môn hoặc GV → bảng lọc ngay
2. **Niên khóa:** Chọn từ dropdown (gợi ý từ DB + sinh tự động đến năm hiện tại+5), hoặc nhập tay ở ô bên dưới
3. **Mở lớp mới:** Click **＋ Thêm** → điền thông tin → **💾 Ghi**
4. **Hủy lớp:** Chọn dòng → **🚫 Hủy lớp** (đánh dấu HUYLOP=1, không xóa vật lý)
5. **Lịch sử:** Click **🕓 Lịch sử** để hoàn tác

#### Nhập Điểm (`/nhapdiem`)
1. Chọn Niên khóa (dropdown gợi ý), Học kỳ, Môn học, Nhóm → **▶ Bắt đầu**
2. Bảng sinh viên tải ra với các ô Điểm CC, GK, CK
3. **Điểm HM tự tính realtime:** `CC×0.1 + GK×0.3 + CK×0.6` (hiện màu xanh)
4. Nhập xong → **💾 Ghi điểm**

### 3.3 Nhóm SV (Sinh Viên)

#### Đăng Ký Lớp Tín Chỉ (`/dangky`)
1. Chọn **Niên khóa** từ dropdown (hoặc nhập tay) và **Học kỳ**
2. Danh sách LTC tự tải sau 400ms — không cần bấm nút tìm kiếm
3. Click **Đăng ký** ở lớp muốn đăng ký
4. Lớp đã đăng ký hiển thị badge **✓ Đã đăng ký** — không thể đăng ký trùng

#### Xem Bảng Điểm (`/phieu_diem`)
- Hiển thị toàn bộ điểm các môn đã học
- Điểm HM ≥ 5: màu xanh (đạt) · < 5: màu đỏ (không đạt)

## 4. Tính Năng Lịch Sử & Hoàn Tác

### Cách sử dụng
1. Click **🕓 Lịch sử** trên bất kỳ trang CRUD nào
2. Modal hiện ra danh sách 10 thao tác gần nhất (mới → cũ)
3. Click **Hoàn tác** ở hành động muốn đảo ngược → xác nhận → trang tự reload

### Quan hệ cha–con trong cây lịch sử
| Thao tác | Hoàn tác tương ứng | Cảnh báo |
|---|---|---|
| Thêm Lớp | Xóa lớp vừa thêm | Nếu đã thêm SV vào lớp → cần xóa SV trước |
| Xóa Lớp | Khôi phục lại lớp | — |
| Thêm SV vào Lớp X | Xóa SV đó | SV là con của Lớp |
| Mở Lớp TC | Hủy lớp TC đó | — |
| Hủy Lớp TC | Mở lại lớp TC | — |

> **Lưu ý:** Lịch sử chỉ tồn tại trong phiên làm việc (session). Đăng xuất → mất lịch sử.
